## 飞扬路由器认证程序

1. 可以突破共享限制,使任何路由器使用校园网
2. 因为使用手机端协议,所以现在闲时(路由器没有人链接)的时候,会被校园网踢下线,你可以在宿舍丢一台电脑保持WiFi一直链接路由器,或者使用openwrt固件的路由器编写一个定时ping脚本
3. 需要自己抓包拿到当天的登陆密码 填入config.ini对应的日期中 ,你可以添加我的机器人 842446224 计算当天的密码 发送**/fy**触发指令
